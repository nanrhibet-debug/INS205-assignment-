Project Description

This single-document interface serves as the primary practical continuous assessment submission for the INS205 (Introduction to Information Systems & Web Technologies) module.

The explicit intent of this project is to showcase strict adherence to semantic web markup validation rules using pure native HTML5 elements, bound tightly with internal cascading style sheet abstractions to handle typography, presentation logic, and device scaling.
